One-Page Technical Flow

Caller Audio In -> Grounded Policy Decision -> ElevenLabs Voice Back Out

This page only shows the live execution path. Audio comes in from browser or phone, gets transcribed by OpenAI Realtime STT, is assembled into a grounded turn, then our backend decides the next line using deterministic policy plus OpenAI chat. Only after that exact line is approved do we call ElevenLabs to speak it back.

Thinking functions: create_language_coach_review() -> run_chat_agent_turn() -> llm_collections_turn() / generate_collections_reply() Truth source: GROUND_TRUTH.md + build_grounded_context() from sap_mock.json Voice back path: assistant_text -> prepare_tts_text() -> ElevenLabs stream -> browser or Exotel playback
